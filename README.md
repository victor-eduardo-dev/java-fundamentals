# Java Fundamentals

Repositório criado para documentar minha evolução nos estudos de Java.

## Sobre

Este repositório reúne exercícios, desafios e pequenos projetos desenvolvidos durante meus estudos de Java.

O principal objetivo é registrar minha evolução ao longo do tempo, mantendo as soluções o mais próximas possível do conhecimento que eu possuía no momento em que cada exercício foi realizado.

## Conteúdos Estudados

### Fundamentos da Linguagem

* Variáveis e Constantes
* Conversão de Tipos
* Entrada e Saída de Dados
* Operadores Aritméticos
* Operadores Lógicos
* Operador Ternário
* Métodos

### Desafios

* Conversão de String para Double
* Operadores Aritméticos
* Operadores Lógicos
* Desafio Final do Módulo (Calculadora)
